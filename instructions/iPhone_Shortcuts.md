# iPhone 快捷指令 (Shortcuts) 配置指南

通过此快捷指令，当您收到包含“验证码”字样的短信时，手机会自动将验证码发送到您的服务器。

### 设置步骤：

1.  **打开“快捷指令” App**，切换到“自动化”标签页。
2.  点击 **“创建个人自动化”** 或右上角的 **“+”**。
3.  选择 **“信息”**：
    *   **发件人**：任意（或留空）。
    *   **内容包含**：输入“验证码”（可以多设几个，如 "code"）。
    *   **确认后运行**：取消勾选，选择 **“立即运行”**（为了更丝滑）。
4.  点击 **“下一步”**，选择 **“新建空白操作”**。
5.  添加操作 **“获取文本中的验证码”**：
    *   输入选择“快捷指令输入”中的“信息”。
6.  添加操作 **“获取 URL 内容”**：
    *   **URL**：输入插件中显示的专用 API 链接（例如 `http://lly.winorgohome.top:8000/push/你的Token`）。
    *   **方法**：选择 **POST**。
    *   **请求体**：选择 **JSON**。
    *   **添加新字段**：
        *   `code`：选择上一步获取的“验证码”。
        *   `device_id`：(可选) 如果想固定发给某台电脑。
7.  点击 **“完成”**。

现在，每当收到短信验证码，它就会自动出现在您的浏览器输入框中了！
